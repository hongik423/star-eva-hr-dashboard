로컬(localhost)에서 로그인 화면이 나오게 하려면:

1. hr-dashboard-deploy 폴더에 .env 파일을 만드세요.
2. 아래 한 줄을 넣고 저장하세요 (비밀번호는 원하는 값으로 바꿔도 됨).

   VITE_APP_PASSWORD=ndh202544*

3. 터미널에서 개발 서버를 다시 실행하세요.
   npm run dev

이후 http://localhost:5177/ 접속 시 비밀번호 입력란만 보이고, 입력하면 대시보드가 열립니다.

※ Vercel 경고 "might expose sensitive information to the browser"는, VITE_ 로 시작하는 변수는 빌드 시 클라이언트에 포함된다는 뜻입니다. 내부용 대시보드 비밀번호로만 쓰면 보통 허용하는 수준입니다.
